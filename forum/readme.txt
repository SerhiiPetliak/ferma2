Установка:

- Закачиваем содержимое архива на хостинг
- Создаем базу данных для форума
- Заходим через браузер в директорию форума
- Запускается установщик, Вам необходимо следовать инструкциям

Данная сборка включает:

- Русификацию
- Файл robots.txt
- Manage attachments ACP module for phpBB3
- NV advanced last topic titles
- Auto Image Resizer
- SiteMap Generator
- First post on every page
- User Reputation Points
- Prime Trash Bin
